METADATA CREATION
The San Diego Historical Society used CONTENTdm to create and store metadata records for their images. Sixteen metadata elements were used and mapped to Dublin Core.

During the image selection process, the SIO Archives used a MS Access database to enter basic descriptive and administrative metadata for each item. Fifty-six metadata elements were adopted and now serve as the standard for the description of digital objects for the SIO Archives. After receiving the scanned images from Luna and the metadata was populated, the database was exported to ASCII delimited text and imported into CONTENTdm. Of the fifty-six metadata elements, only eighteen were selected for display to the public on this site.
Details
To unify these two collections, the SIO Archives Archivist and the San Diego Historical Society Project Archivist created a list of fourteen broad subject heading terms and assigned one or more terms to each image:
  1. Aerial Views
  2. Beaches
  3. Diving
  4. Events
  5. Fishing
  6. Fishing Industry
  7. Harbors
  8. Navigation And Communication
  9. Ocean Life
  10. Ocean Resources
  11. Oceanography
  12. People
  13. Scientific Equipment
  14. Vessels


This list was developed using the Thesaurus for Graphic Materials (TGM), using headings from TGM I and II. If TGM did not provide the Archivists with what they needed, a heading was made up (for example: TGM uses the term BOATS, which was considered too narrow, thus the term VESSELS was chosen). These fourteen categories (full LSTA thesaurus PDF) serve as a predefined search which can be used to query one collection or both simultaneously.

The metadata standards for this project rely on existing standards where applicable, including:

Type: Internet Media Types
http://www.iana.org/assignments/media-types/

Source Type: DCMI Type Vocabulary
http://dublincore.org/documents/dcmi-type-vocabulary/

http://www.alexandria.ucsb.edu/

_ GDC Cruise Index
http://gdcmp1.ucsd.edu/gdc/cruises/cruise.index
_ Phyllis Helm authority list

_ ROGER authority file
_ UNOLS:
http://www.unols.org/images/ships/shipimages.html

Underwater Features: GEBCO Gazetteer of Undersea Feature Names
http://www.ngdc.noaa.gov/mgg/gebco/underseafeatures.html

Subjects: Keywords from Library of Congress Subject Headings controlled vocabulary
